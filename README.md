Online_PCA
==========

Códigos para a aplicação de PCA Online e suas variantes.

Implementação do algoritmo desenvolvido por E. Schlösser, D. Saad e M. Biehl  em

http://www.researchgate.net/publication/230970060_Optimization_of_on-line_principal_component_analysis


****************************************************************

Codes for the application of an Online PCA method and it's varieties.

Implementation of the algorithm developed by  E. Schlösser, D. Saad and M. Biehl in

http://www.researchgate.net/publication/230970060_Optimization_of_on-line_principal_component_analysis
